Данная программа представляет из себя файловый менеджер, который позволяет только перемещаться по папкам. Для запуска введите в терминале "make && ./main.exe".

Управление:
w - переместиться на строку выше
s - переместиться на строку ниже
a - выбрать левое окно для работы
d - выбрать правое окно для работы
e - переместиться в выбранную папку
q - выйти из программы
